# 🩸 BloodConnect

**BloodConnect** bridges the critical gap between blood donors and healthcare facilities by providing an intuitive, secure platform for managing blood donation campaigns, donor registration, and inventory tracking. Our mission is to save lives by making blood donation more accessible and efficient.

![BloodConnect Cover](https://images.unsplash.com/photo-1615461066841-6116e61058f4?q=80&w=2783&auto=format&fit=crop)

---

## ✨ Key Features

- **Donor Registration & Management**: Easy signup process with medical history tracking and eligibility verification.
- **Blood Bank Integration**: Real-time inventory management for blood banks and hospitals.
- **Donation Scheduling**: Smart appointment booking system with automated reminders.
- **Emergency Alerts**: Urgent blood requirement notifications sent to eligible donors.
- **Campaign Management**: Tools for organizing blood donation drives and events.
- **Analytics Dashboard**: Comprehensive reporting for donation trends and impact metrics.
- **Mobile Responsive**: Optimized experience across all devices.
- **Multi-language Support**: Accessible to diverse communities.

---

## 🛠️ Tech Stack

This project is built as a single-page React application for extreme simplicity and portability.

- **Frontend Framework:** React 18 (via CDN)
- **Styling:** Tailwind CSS (via CDN)
- **Icons:** Lucide React
- **Compiler:** Babel Standalone

## 🚀 Getting Started

Since this project relies on CDN links, setting it up is incredibly fast and simple! No `npm install` required.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/LaveenChordia/Blood-Donation-Website.git
   cd Blood-Donation-Website
   ```

2. **Run the Application:**
   Simply open the `blood.html` file in any modern web browser.
   
   Alternatively, you can use a local server for a better development experience:
   ```bash
   # If you have Python installed:
   python3 -m http.server 8000
   ```
   Then navigate to `http://localhost:8000/blood.html`
